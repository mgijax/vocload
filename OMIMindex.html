OMIM Vocabulary Load Data and Logs

OMIM Vocabulary Load (vocload) Data, Runtime Files, Reports, and Log Files

Inputs

  • download directory - original downloaded files
  • omim.txt - OMIM input file; copy from download directory
  • OMIM.translation - OMIM-to-MGI Term Translation fil; maintained by A&P
  • OMIM.special - OMIM-to-MGI Word Translation fil; maintained by A&P
  • OMIM.synonym - OMIM synonyms file; maintained by A&P
  • OMIM.exclude - OMIM exclude file; maintained by A&P

Reports

  • discrepancy.html - Curator Report
  • OMIMtermcheck.current.rpt - Checks new OMIM.tab terms vs. MGI OMIM terms; new OMIM.tab OMIM secondary ids vs. MGI OMIM secondary ids

Outputs

  • fullLog.txt - File generated by vocload containing a complete record of the job, including all SQL executed.
  • OMIM.tab - File generated by vocload containing all the terms being processed.
  • All available run-time files - All files located in the run-time directory.
